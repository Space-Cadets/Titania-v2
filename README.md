# Titania-v2

Created with create-react-app

### Development
To start development server:

```sh
npm start
```

To build for production:

```sh
npm run build
```

### Setting up Nuclide dev environment
1. Install Atom https://atom.io/

```sh
apm install nuclide
brew update
brew install watchman
brew install flow
apm install linter-eslint
apm disable linter
```
